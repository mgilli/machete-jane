# machete-jane
A 2D platformer game made with Pygame

Requires Python, as well as pygame and pytmx

Arrows to Move (or WASD on a keyboard set to QWERTY)
Space to Jump
Left ctrl or Left alt to Shoot


Art and Music made by Mathias Gilli
